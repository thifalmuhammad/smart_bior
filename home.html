Masalah & Solusi

Dari pengamatan manual ke data yang siap dipantau.

Metode manual seperti pengamatan visual dan pengambilan sampel laboratorium sering lambat, subjektif, dan sulit dilakukan terus-menerus. Smart-Bior merapikannya menjadi pembacaan sensor yang tersimpan otomatis.

01

Pengukuran objektif

Data OD dibaca dari sensor optik, bukan hanya warna kultur yang dinilai secara visual.

02

Realtime dashboard

Nilai terbaru dan riwayat pengukuran tampil di dashboard web untuk pemantauan harian.

03

Lebih terjangkau

Dirancang sebagai alternatif praktis untuk pemantauan rutin tanpa instrumen lab yang mahal.

Fitur Utama

Ringkas, otomatis, dan mudah dibaca.

OD

Optical Density

Memantau kepadatan kultur Spirulina sebagai indikator pertumbuhan biomassa.

IoT

Firebase Realtime

Data dari perangkat dikirim ke Firebase dan dibaca langsung oleh dashboard web.

°C

Suhu Kultur

Nilai suhu ditampilkan bersama OD agar kondisi kultur lebih mudah dievaluasi.

UI

Riwayat Data

Grafik dan tabel membantu melihat pola perubahan dari waktu ke waktu.

Alur Kerja

Empat langkah pemantauan.

Sensor membaca kultur

Perangkat mengambil nilai OD dan suhu dari sistem bioreaktor.

Data dikirim ke cloud

ESP32 mengirim data ke Firebase Realtime Database.

Dashboard menerima event

Halaman web memperbarui tampilan saat data Firebase berubah.

Operator membaca tren

Riwayat dan grafik digunakan untuk mengambil keputusan budidaya.
